- `product_id`: `4564c6df-65d8-4c2b-9797-c8a64944c6c5`
- `draft_id`: `3f1d87d2-3667-44e1-b67a-3638d2abf413`
- `campaign_link_id`: the campaign link used
- `channel`: the actual channel used
- `campaign_name`: matching campaign name from the table above
- `clicks`: real click count only
- `conversions`: real conversion count if visible, otherwise `0` or blank depending on certainty
- `revenue`: real revenue only if reported by Systeme.io, otherwise blank
- `recorded_at`: date/time the numbers were checked
- `notes`: include source of the measurement, e.g. `Systeme.io dashboard`, `manual blog analytics`, `Medium stats`, or `LinkedIn post analytics`

1. Share manually in one controlled channel only, preferably manual blog or private test.

3. Record clicks, conversions, and revenue only from real dashboards.
4. Add notes describing where the numbers came from.
5. Review Command Center, Performance Insights, and Data Quality Center.
