## באג 5: Claude לא מפרסם פוסטים

- למה שבור: Claude מנסה לפרסם דרך טאבים (שלא עובדים — באג 2) במקום דרך API
- מה Claude עושה לא נכון: חוזר על אותו ניסיון כושל (פתיחת טאבים) במקום לבנות פרסום דרך API
- תיקון נדרש: לבנות מנגנון פרסום דרך API לכל פלטפורמה:

  - LinkedIn: OAuth App → Share API

  - TikTok: ידני (אין API לפרסום טקסט)
  - Quora: ידני (אין API פתוח)
  - Reddit: OAuth App → Submit API
- חוסם: Claude לא בנה את מנגנון ה-API עדיין
